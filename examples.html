Commissions

All commissions are divided by 10^8.
Actual commission is SendTx * 10^8 = 100 000 000 000 PIP = 0,0000001 BIP.

  • SendTx = 1000
  • ConvertTx = 10000
  • CreateTx = 100000
  • DeclareCandidacyTx = 100000
  • DelegateTx = 10000
  • UnbondTx = 10000
  • RedeemCheckTx = 1000
  • PayloadByte = 500
  • ToggleCandidateStatus = 1000

Wallet

Generate wallet

Generates wallet from a random BIP39 mnemonic phrase (uses bip39.generateMnemonic under the hood).
So wallet instance will contain:
  • Private key - random 64 bytes
  • Mnemonic phrase - random 12 words

Export wallet

There are 3 ways of exporting a wallet:
  • Export Private key - random 64 bytes
  • Export Mnemonic phrase - random 12 words
  • Export Vault file

Import wallet

There are 3 ways of importing a wallet:
  • Import Private key - random 64 bytes
  • Import Mnemonic phrase - random 12 words
  • Import Vault file
